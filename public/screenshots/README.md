# Kanyini Connect Screenshots

This folder contains screenshots of the Kanyini Connect platform for display on the website.

## Required Screenshots

Please save the following screenshots in this folder:

1. **community-feed.png**
   - The Community Feed view showing posts, filters, and activity stats
   - Recommended size: 600x1200px (mobile view)

2. **project-detail.png**
   - The Project Details page showing funding progress, team status, and project information
   - Recommended size: 600x1200px (mobile view)

## Format
- Format: PNG
- Quality: High resolution for retina displays
- Background: Include the full app interface

The screenshots will be displayed in a beautiful grid layout with hover effects and feature tags on the `/kanyini-connect` page.
